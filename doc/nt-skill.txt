[270] 关于fy和nt的武功mix套路之moon篇          玉罗刹(yuluosha)(Mon Feb 17)

国际惯例先做个广告，更多nt介绍参见上回和上上回，分别read 263,read 269。
下面介绍一下fy和nt的武功mix套路，本期针对moon门派。
moon的优点你懂的，几乎是fy防御力最强的门派，免伤，断pfm，反击伤害...
至于输出moon也不弱，当然我们要考虑怎么让他最强。
moon的缺点你也懂的，没有空手技能，气血短，needle耗费太厉害。
我们的目标就是扬长避短，将fy强大的副手系统和nt的武功结合起来。
公共说明：
1.moon的心法和步法是必学的；
2.fy的副手pfm只能用fy自订武器；
3.nt技能放副手只能有普通攻击，不能自动pfm；
4.nt的技能pfm只能用nt的武器，fy自订不能pfm。
mix玩法之一：双刀客
fy技能：moon的门派心法 + 步法 + 一刀流
nt技能：五虎断门刀 + 狂风刀法 + 一指禅
主手装备nt的高杀伤武器，副手装备fy自订
主手pfm：五虎断门刀，pfm duan
副手pfm：一刀流，百鬼
普通攻击：狂风刀法
空手技能：一指禅
玩法介绍：
1.set secon_perform hyakki这个固定，副手pfm用百鬼。
2.开杀时等副手出百鬼，出了百鬼后enable blade 狂风刀法，用狂风刀法的
普通攻击，能出百鬼的时候（比如杀死一个npc获取1点杀气或战斗超过2分钟）
再enable一刀流。这些用tri很容易实现的。
3.根据战斗情况主手用五虎断门的pfm，pfm做个简单alias即可。
4.正常情况刀法enable是狂风刀法。
5.对手兵器杀伤高的就用一指禅碎掉他的兵器。自己兵器被切就用一指禅战斗。
各种nt武功入手方式：
五虎断门刀：北京找陈近南加入天地会，然后扬州找李世力学习，学完可以找
陈近南退会，无任何副作用。该刀法级别越高pfm威力越大。
狂风刀法：找nt高手帮你杀boss田伯光，随机掉落秘笈，该刀法普通攻击有杀
伤特效。秘笈读到200级，建议res到350级，平砍威力最大。
一指禅：杭州找陈家洛学150级指法，少林藏经阁拿少林wuji，读一指禅到100
级，res到120级出pfm jingmo，150级出pfm dian。
说明：该玩法入手门槛较低，适合所有级别的moon玩家。
mix玩法之二：双剑双修
fy技能：moon的门派心法 + 步法 + 一刀流
nt技能：破阳冷光剑或祝融剑/紫盖/芙蓉/天柱/石廪任一剑法，用来普通攻击
        金蛇剑法，用来pfm
        一指禅，空手
主手装备nt的高杀伤武器，副手装备fy自订
主手pfm：金蛇剑法，pfm duo, kun, kuangwu, shi, suo, wan, zhui
副手pfm：一刀流，百鬼
普通攻击：破阳冷光剑或祝融剑/紫盖/芙蓉/天柱/石廪任一剑法
空手技能：一指禅
玩法介绍：
金蛇剑法非常酷，缺点是平砍会自动pfm 9剑，极耗内力，fy那点内力给本打不了
几个回合，所以只用它的pfm。普通攻击随便用一种剑法就行。
金蛇秘笈入手：100左右的号去华山秘洞里挖金蛇秘笈，等级低了会jump死，金蛇
剑法的秘笈也可以让高手帮你去拿，可读到120，res到200出全所有pfm。
说明：该玩法的主要难度就是金蛇剑法的获取和res，适合120+的玩家，有兴趣
解谜学习的盆友可以咨询。
mix玩法之三：剑法名家
fy技能：moon的门派心法 + 步法 + 一刀流
nt技能：衡山五神剑 + 狂风刀法（选） + 一指禅
主手装备nt的高杀伤武器，副手装备fy自订
主手pfm：衡山五神剑，pfm hui, qian, shen(max_neili>5500)
副手pfm：一刀流，百鬼
普通攻击：衡山五神剑或狂风刀法
空手技能：一指禅
玩法介绍：
主手pfm改成衡山五神剑，其他和上面类似。
说明：衡山五神剑入手难度较高，需要满足一定条件，并且需要大量pot用来res，
该玩法有一定门槛，适合139+的玩家，有兴趣解谜学习五神剑的盆友可以咨询。
mix玩法之四：偷师少林
nt的少林是个很悲催的门派，好技能一大堆，但基本上都可以被别的门派偷学。

3.玄冥神掌：该技能也是free技能，我觉得威力应该很强大，但是自己没试过，有兴

4.宗师武功：如果fy玩家修炼到很高的级别，兵器、空手、内功、轻功各项技能达到

现在还玩mud的大都是10多年前的老mud玩家，比较怀旧，玩玩mud打发空闲时间，
所以大部分人都不太喜欢接收新鲜事物，尤其是当觉得目前的东西还比较好玩

去玩了下nt系统，结果越玩越觉得nt可玩性很高，现在爱不释手深陷其中。其
实对于一个游戏来说，玩法多一些对玩家总是好事，你要是接受不了可以玩自

大家多一些选择，特别是让那些无聊的老玩家有新的动力。
有人就说了，那干脆去玩纯nt的站算了。呵呵，那区别可大了。第一，纯nt的
站点几乎全是挂机的robot，毫无生气，因为我们这是mix版本，对nt做了很大
的限制，所以我们这还是鼓励手动为主，手动才是乐趣之所在。第二，在这个
mud玩的都是对fy有感情的玩家，不然早去别的mud了，对这里的服务器、玩
家、wiz都比较熟悉，特别是老玩家，一般也不会轻易更换阵地。
之所以最近不断给大家推荐nt跟fy mix的方式方法，推荐nt的部分武功，就是
希望大家可以尝试一下，从fy渐渐接触nt，了解nt，当你喜欢上之后甚至可以
拜入nt的门派深入尝试。
老有人说纯fy没了没法完了，这说法欠妥，fy的核心内容从来没变过，你不喜
欢nt这些可以不玩，甚至可以把nt各种info屏蔽掉，你玩你的fy不会受任何影
响。不能因为你不喜欢就不给别人选择的权利。
更不要担心我们这个fy游戏最后变成nt世界了，那是不可能发生的。wiz对这
块管控的很严，所有的战斗系统、精气神内力属性各参数、以及各种标准都
是按照fy的系统设定，核心内容从现在到将来都会一直是fy的东西。nt的引入
Γ约叭媚切┒詅y失去兴趣的老玩家有
新的东西可以玩。长远以后甚至会考虑nt和fy的进一步融合，打造独一无二的
mix版本，既保持fy核心又有多种玩法，你在外面的野战是体会不到这些的。
二.fy玩家发展思路 - 稳扎稳打型，非快餐速升版
fy老玩家一抓一大把，我的思路也不见得最好，但是根据我玩游戏的特点，我
对所有新引入的内容比如宠物、副手、HC这些都会深入研究一下，所以我的
思路可以对大家形成参考，说不定可以少走弯路。
1.人物选择HC模式，HC的优势你懂的。
2.土豪前期可以只做古龙带路任务，其他一律cancel，包你速度到30+。
3.30+随便学点技能，把宠物练起来，守经楼。其实有强力宠物的话，100级守
经楼都没问题，一般人我都不说，grin...
4.守完经楼在宠物配合下一路quest，每升10级练下宠物，应该无任何风险。
5.80+开始ability，稳扎稳打，这点很重要，一定要打好基础。
6.每升10级左右刷一会大旗谷换装备，主要是加精气神命中精准暴击闪躲招架
这些属性，符文随便插插，都是过程装备没必要练高级别。刷大旗谷的同时攒
宠物币。
7.副手百鬼适合任何门派，fy杀伤最高的副手pfm，没有之一。
8.如果打算后期学习nt技能的，在139之前可以去杀森林副本攒体会，汲取为
武学修养，根据自己的武功和装备情况，再差的120+也应该能杀了。
9.139以后的升级套路分为三种：
    第1种，打算做普通quest的，建议将级别控制在145-149，这个阶段的
quest又丰富又好做，tri高手甚至可以实现全自动；
    第2种，不打算做普通quest的，可以继续升级超过150，建议控制在200以
下，再高了maze难度大幅增加。超过150级以后，玩家的天机任务将只有小鱼
儿、古龙、运镖、maze，没有普通quest。
    第3种：打算只杀倭寇岛的，可以将级别无限提升，提升方式有杀nt的副
本，做nt能做的任务等。但是fy玩家级别太高不建议，在两边都不好混。
总结：
fy玩家发展套路 - quest -> 宠物 -> 副手 -> ability -> maze -> 139 ->
3p -> 4p -> 5p ->6p ->7p -> 倭寇岛（终极追求）
fy玩家终极修炼之1：
最高级别149，任务内容：普通quest + 小鱼儿 + 古龙 + maze(1-7p) + 倭寇
岛（有强力装备支撑）
fy玩家终极修炼之2：
最高级别200，任务内容：小鱼儿 + 古龙 + 运镖 + maze(1-7p) + 倭寇岛(有
装备支撑）
fy玩家终极修炼之3：
最高级别无限，任务内容：maze(1p2p3p) + 倭寇岛（容易）
